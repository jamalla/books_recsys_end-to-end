# Books Recommendation Systems (End-to-End)

An end-to-end machine learning project for building a book recommendation system. This project implements a complete ML pipeline including data ingestion, validation, transformation, and model training to provide personalized book recommendations based on user ratings and preferences.

## Features

- **End-to-End ML Pipeline**: Structured pipeline with data ingestion, validation, transformation, and model training stages
- **Modular Architecture**: Organized codebase with separate components for each pipeline stage
- **Streamlit Interface**: Interactive web application for book recommendations
- **Docker Support**: Containerized deployment with Dockerfile
- **Configurable**: YAML-based configuration management

## Tech Stack

- Python 3.11+
- scikit-learn
- pandas
- numpy
- Streamlit
- PyYAML
